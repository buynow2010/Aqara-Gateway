# 更新日志

所有重要的项目变更都将记录在此文件中。

## [0.3.2] - 2025-11-20

### 新增 ✨

- 完整中文化所有用户界面
- 添加完整的中文 README 文档
- 添加 HACS 配置支持
- 添加 .gitignore 文件

### 汉化内容 🌏

#### 配置界面
- 汉化 strings.json 中的所有配置文本
- 汉化 translations/zh-Hans.json 翻译文件
- 配置流程描述、字段名称全部中文
- 系统健康信息中文化

#### 设备状态
- **门锁** (`core/lock_data.py`)
  - 所有锁通知信息汉化
  - 门状态：门已打开、门已关闭、门未关好等
  - 锁状态：已上锁、已解锁、已反锁等
  - 解锁方式：通过密码、指纹、NFC、人脸、HomeKit 等解锁
  - 离家模式、电池通知、验证失败等状态
  
- **传感器** (`sensor.py`)
  - "Unknown" 改为 "未知"
  - 门锁传感器初始状态汉化

- **二进制传感器** (`binary_sensor.py`)
  - 按钮默认值 'unknown' 改为 '未知'
  - 振动、魔方动作默认值汉化

#### 设备动作（`core/const.py`）
- **按钮动作 (BUTTON)**
  - 单击、双击、三击、四击
  - 长按、释放、摇一摇
  - 顺时针、逆时针
  - 长按顺时针、长按逆时针
  - 旋转、长按旋转、多次

- **双键按钮 (BUTTON_BOTH)**
  - 单击、双击、三击、长按、释放

- **振动传感器 (VIBRATION)**
  - 振动、倾斜、跌落

- **魔方动作 (CUBE)**
  - 翻转90度、翻转180度
  - 移动、敲击、摇晃
  - 旋转、长按、警报

- **其他状态**
  - 门铃状态："door_bell" 改为 "门铃"
  - 窗帘状态："stop" 改为 "停止"，"hinder_stop" 改为 "阻挡停止"

#### 选择项（`core/utils.py`）
- **风扇模式**: 低风、中风、高风
- **摆风模式**: 启用、禁用
- **运行模式**: 制暖、干燥、风扇、排气
- **监控模式**: 无方向、左右
- **接近距离**: 近、中、远
- **清洁方向**: 关闭、自动、手动
- **水温**: 常温、31°C、33°C、35°C、37°C、39°C
- **清洗等级**: 弱、中弱、中、中强、强
- **暖风等级**: 关闭、常规、低、中低、中、中高、高
- **默认选项**: 关闭、开启

#### 窗帘设备（`cover.py`）
- **充电状态**: 未充电、充电中、停止充电、充电失败
- **电机行程**: 无行程、行程已设置
